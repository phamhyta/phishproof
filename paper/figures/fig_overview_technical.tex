\caption{\toolname{} technical pipeline. $C_1$ elicits structured agent outputs:
each agent plans over page observables, calls verification tools, emits
$(y_i,\evset_i)$, and contributes typed nodes to the evidence graph $H(\page)$.
$C_2$ value-normalizes nodes and forms the majority-supported consensus subgraph
$\cons(\page)$, yielding evidence agreement $\agr$. $C_3$ re-checks each
consensus node with its type-specific verifier $\gver$ to compute groundedness
$\grd$. $C_4$ combines the two as $\geascore=\agr\cdot\grd$, maps it through a
delayed-oracle calibrator $\kappa$, and returns either $(\hat y,\hat\rho,\cons)$
or an abstention $\bot$.}
\Description{A four-stage technical pipeline with panels C1 through C4. The
figure shows agent evidence elicitation, value-normalized evidence consensus,
tool-grounding, calibrated selection, side inputs from verification tools and
delayed oracle labels, and the final act or abstain output.}
\label{fig:overview}
